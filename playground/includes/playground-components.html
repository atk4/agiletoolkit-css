Swatches

.swatch-* component can be assigned to any element. Text color will adapt based on luma value (perceptual brightness) of the swatch. You can easily create as many swatches as you need using .createSwatch() mixin (see variables.less).

.items-spacing is a component that creates margins between its children. Make sure there are no spaces between your action buttons.

Separator

.margin-left--xs.margin-right--xs is a component that creates margins between its siblings. It is recommended to use it with <span> tag.
